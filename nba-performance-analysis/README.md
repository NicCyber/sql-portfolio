# NBA Performance Analysis

## Overview

Used SQL to analyze NBA game statistics across multiple seasons to identify scoring trends, home-court advantage, and the relationship between three-point shooting and winning.

## SQL Skills

- AVG()
- COUNT()
- MIN()
- MAX()
- GROUP BY
- HAVING
- ORDER BY

## Questions Explored

- How has scoring changed over time?
- Has home-court advantage changed?
- Does better three-point shooting correlate with winning?
- Which teams consistently shoot well from three?
