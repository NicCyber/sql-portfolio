# London Transit Analysis

## Overview

Analyzed London Underground ridership data to identify travel patterns, commuting behavior, and peak transit periods.

## SQL Skills

- GROUP BY
- SUM()
- ORDER BY
- Multi-column grouping
- Aggregate functions

## Questions Explored

- Which time periods are busiest?
- Which travel purposes are most common?
- How does travel vary by zone?
- How do commuting patterns change throughout the day?
